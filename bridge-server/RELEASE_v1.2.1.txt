To create the GitHub release and trigger builds, run:

OPTION 1 - Using GitHub CLI:
./.github/create-release.sh v1.2.1

OPTION 2 - Using GitHub API:
GITHUB_TOKEN=your_token ./.github/create-release-api.sh v1.2.1

OPTION 3 - Manual (via web UI):
1. Go to: https://github.com/soundrolling/electrondante/releases/new
2. Select tag: v1.2.1
3. Title: Dante Audio Client v1.2.1
4. Copy release notes from: RELEASE_NOTES_v1.2.1.md
5. Click "Publish release"

This will trigger GitHub Actions to build:
- macOS (Intel + Apple Silicon)
- Windows (x64 + x86)
- Linux (x64 AppImage)

Monitor build: https://github.com/soundrolling/electrondante/actions

What's Fixed in v1.2.1:
- Fixed V8 JavaScript Out of Memory (OOM) error on startup
- Increased memory limit to 4GB
- Added initialization retry limits
- Improved error diagnostics
- Optimized memory usage
